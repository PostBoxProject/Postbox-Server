# 💌 친구들에게 익명 편지를 남길 수 있는 서비스 💌

<br>

## 📮 기능 소개

  - 자신만의 우체통을 생성해 편지를 받아요!
  - 친구의 우체통에 편지를 남겨 보세요!
  - 다른 사람의 우체통에 남겨진 편지를 구경해요!
